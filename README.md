# EasyHire

EasyHire is a web application that connects employers and workers.

## Technologies Used

* Java
* Apache NetBeans
* MySQL
* Maven

## Features

* Employers can post job requests
* Workers can find nearby jobs
* Easy hiring process

## Database Setup

Import the following SQL files in MySQL:

* mydb_worker.sql
* mydb_job_requests.sql

## How to Run

1. Open the project in Apache NetBeans
2. Import SQL database
3. Run the project
